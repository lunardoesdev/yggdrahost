# yggdrahost

I want to have multiple sites and bots to be hosted as mountable
apps on same server in the same process.

I use bun, elysiajs.

I use modular elysiajs servers and the main router that.

# deps

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.10. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
